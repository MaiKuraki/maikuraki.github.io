# 结论

在我们日常开发中，我们可能会需要在某些场景中实时同步一些数据，比如同步至 Android Native 或 iOS Native，亦或在网络游戏中需要一些高频率同步的数据，此时数据压缩与解析所占用的性能开销是我们需要注意的部分，JSON 在这种场景就不应该使用，此时 Protocol Buffers 或 FlatBuffers 都是推荐使用的方案。
